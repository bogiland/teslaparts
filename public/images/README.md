# Папка для ваших изображений

Сюда вы можете загружать свои собственные фотографии для товаров.
Код уже полностью настроен! Как только вы загрузите сюда картинку с нужным названием, она автоматически заменит временную картинку на сайте. Никакой код менять не придется.

## Список файлов, которые нужно загрузить в эту папку:

*   `bumper.jpg` — Передний бампер Tesla Model 3
*   `wheels.jpg` — Аэродинамические диски 18" Model 3/Y
*   `yoke.jpg` — Штурвал (Yoke) Tesla Model S/X
*   `console.jpg` — Накладка на центральную консоль
*   `charger.jpg` — Зарядная станция Wall Connector
*   `keycard.jpg` — Ключ-карта (Key Card)
*   `screen-protector.jpg` — Защитное стекло для экрана
*   `filter.jpg` — HEPA фильтр салона
*   `mats.jpg` — Всепогодные коврики салона
*   `brakes.jpg` — Комплект тормозных колодок
*   `suspension.jpg` — Рычаг передней подвески

### Как это сделать прямо здесь:
1. Слева в панели файлов найдите папку `public`, затем `images`, затем `products`.
2. Нажмите правой кнопкой мыши на папку `products` (или используйте кнопку загрузки) и загрузите свои картинки.
3. Убедитесь, что названия файлов в точности совпадают со списком выше (включая расширение `.jpg`).

*Если ваши картинки в формате `.png`, вы можете либо переконвертировать их в `.jpg`, либо просто один раз поменять `.jpg` на `.png` в файле `src/data/products.ts`.*
